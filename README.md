# ExpenseTracker
Income &amp; Expense Tracker A multi-layered ASP.NET Core web application for managing personal finances.
Users can securely log in, categorize income and expense transactions, and view financial summaries.
Key Features:
	•	User registration and authentication
	•	Create, read, update, delete (CRUD) income/expense records
	•	Categorization (Food, Salary, Investment, etc.)
	•	Daily or Period-Based reports.
